Scope: UPT operator sketch
Branch fit: Plexi‑QFT

“UPT” denotes our family of **Update / Universe Propagation / Timeline** operators that evolve a timeline state in discrete steps.

In this version:
- [EXP] We treat the UPT as an abstract local update rule on a timeline configuration, not fixed to any one model.
- [SPEC] Simple single-bit-flip Markov chains are labeled as **Timeline‑0 toys** and never assumed to be the true UPT.
- [EXP] A real UPT operator must specify: state space, local update rule, causal/UV constraints, and any conserved quantities of interest.

When using this block, assume the UPT is *not yet fixed*; agents should propose scaffolds and ask for the concrete local rule.
