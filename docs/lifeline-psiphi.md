# psiPhi Lifeline v0.1

psiPhi is the central **lifeline framework** of the Symbiopoietrix Lab. It provides a shared coordinate system that links our physics ideas, timeline models, and software artifacts.

At a high level, psiPhi connects:

- **QFT layer**  
  Formal objects such as fields, actions, propagators, Green’s functions, operator algebras.

- **Timeline / UPT layer**  
  Discrete models of history and evolution: timelines, UPT operators, Markov-style views, and finer “timeline morphemes” carved out of multichannel traces.

- **Software / tools layer**  
  Concrete realizations in code: data structures, MCP tools, CLIs, schemas, and logs that implement or probe the above.

The psiPhi lifeline is not a single fixed model. It is a scaffold for:

- Mapping between formal QFT objects and discrete timeline models.
- Anchoring new operators (UPTs) and toy models (Timeline‑0) without losing track of what is “real” vs “placeholder”.
- Giving multiple personas (Plexi‑QFT, Plexi‑Implementor, Orchestrator) a common language when they talk about the same structures from different angles.

As the lab evolves, psiPhi’s lifeline will gain more precise sections (definitions, diagrams, examples), but this file remains the high‑level spine.
