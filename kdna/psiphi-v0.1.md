# K-DNA: psiPhi v0.1

ID: kdna-psiphi-v0.1
Scope: Framework overview
Branch fit: Plexi‑QFT, Orchestrator
Status: EXP (experimental), not canonical

psiPhi is a QFT-flavored, category-tinged **lifeline framework** that links:
- [TXT] formal QFT objects (fields, actions, propagators, Green’s functions),
- [EXP] discrete timeline models (UPTs, Markov views of history, “timeline morphemes”),
- [EXP] software artifacts (code, tools, data structures) used to probe those models.

Its job is to provide a single, coherent spine so that:
- theoretical moves, stochastic models, and implementation details can be mapped to each other,
- multiple LLM personas can talk about “the same object” across math, algorithms, and code.

When in doubt, treat psiPhi as “the shared coordinate system” for our QFT/timeline work, not as a specific model or ontology.
