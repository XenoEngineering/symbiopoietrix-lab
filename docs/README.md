# Symbiopoietrix Lab

Symbiopoietrix Lab is a workspace for human–AI co‑development of theories, tools, and artifacts around the psiPhi lifeline, UPT timelines, and multi‑persona LLM collaboration.

The lab treats:
- **S₀ / S₁ personas** as code-like specs (system prompts with branches),
- **K‑DNA blocks** as short “gene” snippets of domain knowledge,
- **Timeline morphemes** as small, labeled units of temporal work structure (bursts, edits, hops),
- **Orchestrator agents** as routers, not solvers.

The human investigator (you) is always the mission planner and zinker‑persona chooser; agents amplify, refactor, and remember.

## Repository layout

- `docs/` — human-first documentation: overview, manifesto, psiPhi lifeline.
- `kdna/` — versioned K‑DNA blocks (psiPhi, UPT operators, Forge style).
- `personas/` — S₀ core spec and S₁ branch personas (Plexi‑QFT, Plexi‑Implementor, Orchestrator).
- `orchestrator/` — routing rules, examples of task → persona + K‑DNA + U₁.
- `timelines/` — schema and example traces for timeline morphemes (multichannel “ghost goggles” data).
- `tools/` — prompts and integration sketches (MindChat, MCP, other agents).

RSI note: files are kept short and modular so they can be managed primarily by copy/paste and coarse edits, not heavy retyping.
